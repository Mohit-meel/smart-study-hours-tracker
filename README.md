
# Smart Study Hours Tracker

## Overview
Smart Study Hours Tracker is a simple Python-based application that helps students record and manage their daily study hours. It allows users to keep track of their study time and monitor their learning progress.

## Features
- Add daily study hours
- Store study records
- View study history
- Simple and easy to use
